# SEGTRACK
El proyecto SE GTRACK tiene como objetivo desarrollar un sistema web para el control de seguridad física y la trazabilidad de equipos, personal y medios de transporte dentro de una organización.
Este sistema permitirá registrar el ingreso y la salida del personal, así como de los dispositivos tecnológicos que porten y los vehículos que utilicen.
El acceso y la verificación se realizarán a través de códigos QR, lo que facilitará una validación ágil y precisa de la identidad del usuario. Además, el sistema generará alertas de confirmación para garantizar la integridad de los datos registrados.
